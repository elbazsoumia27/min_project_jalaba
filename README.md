# Djellaba - Premium Moroccan E-Commerce Application

A modern, elegant React-based e-commerce application showcasing traditional Moroccan Djellabas with a luxury design aesthetic.

![Djellaba](https://img.shields.io/badge/React-18.2.0-blue)
![Tailwind CSS](https://img.shields.io/badge/TailwindCSS-3.3.0-06B6D4)
![License](https://img.shields.io/badge/License-MIT-green)

## 🌟 Features

### ✨ Core Functionality

- **Modern React Architecture**: Built with Functional Components and Hooks
- **State Management**: Simple and elegant cart system using React `useState`
- **Responsive Design**: Fully responsive on mobile, tablet, and desktop devices
- **Moroccan-Inspired Colors**: Premium color palette (Emerald Green, Gold, Sand)
- **Elegant Typography**: Serif headings with refined sans-serif body text

### 🛒 Components

1. **Navbar**
   - Logo with "Djellaba" branding
   - Navigation links (Home, Shop, About, Contact)
   - Cart icon with dynamic item counter
   - Sticky positioning for easy access

2. **Hero Section**
   - Eye-catching gradient background with decorative elements
   - Compelling headline and description
   - "Shop Now" call-to-action button
   - Premium image placeholder

3. **Product Grid**
   - Display of 6 premium Djellaba products
   - Each product shows:
     - Product image
     - Name
     - Price badge
     - Description
     - 5-star rating
     - "Add to Cart" button
   - Hover effects and smooth animations
   - Responsive grid layout (1 col mobile, 2 col tablet, 3 col desktop)

4. **Cart Sidebar**
   - Slide-out modal with smooth animations
   - Display of all cart items
   - Quantity controls (increment/decrement)
   - Individual item removal
   - Cart total calculation
   - "Proceed to Checkout" button
   - Empty cart state handling

5. **Footer**
   - Company branding and description
   - Social media links
   - Quick navigation links
   - Customer service information
   - Newsletter subscription form
   - Terms and policies links

## 🎨 Design Highlights

## 🚀 Getting Started

### Prerequisites

- Node.js (v14.0.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd min_project_jalaba
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm start
   ```

   The application will open automatically in your browser at `http://localhost:3000`

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build/` directory.
